# FRAGMENTS AI

A powerful AI-powered platform that offers video generation, image creation, and intelligent chat capabilities.

## Features

- 🎬 **AI Video Generator**: Create videos with subtitles and audio from text prompts
- 🖼️ **AI Image Generator**: Generate stunning images using AI (powered by Craiyon)
- 💡 **Viral Social Post Ideas**: Generate engaging social media content
- 🤖 **FRAGMENTS AI Agent**: Chat with your personal AI assistant

## Tech Stack

- **Backend**: Python Flask
- **Frontend**: HTML, CSS, JavaScript
- **AI Integration**: OpenRouter API (GPT-3.5-turbo)
- **Deployment**: Docker, Heroku-ready
- **Styling**: Custom dark theme with modern UI
